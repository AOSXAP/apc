# Foobar

Audio Player written in C, under development - early stage

## Installation

```bash
git clone git@github.com:AOSXAP/apc.git
cd apc
mkdir build && cd build
cmake ..
make
./bin/apc
```

## Usage

```bash
You can use the following commands: "help / stop / resume / queue / play <song> / time ... \n"
```
